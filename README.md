# Turkish Elections Analysis Dashboard (2018-2023)

## Overview
This Power BI dashboard provides a comprehensive analysis and visualization of Turkish election results, comparing data from 2018 and 2023 presidential elections. The interactive dashboard allows users to explore voting patterns, regional distributions, and demographic changes across different electoral periods.

## Features

### Navigation
- Left-side menu for easy switching between with bookmarks:
  - 2018 Election Results
  - 2023 Election Results
  - Comparative Analysis

### Data Visualizations

#### Election Results Maps
- Interactive map of Turkey showing regional voting distributions
- City-level drill-down capabilities
- Color-coded regional results

#### Major Cities Analysis
- Detailed breakdown of election results in key metropolitan areas:
  - Istanbul
  - Ankara
  - Bursa
  - Antalya

#### Comparative Analysis Tools
- Vote share changes between 2018 and 2023
- Regional voting pattern shifts
- Gender distribution analysis of voters
- Voter turnout comparison

### Key Metrics Displayed
- Candidate-wise vote percentages
- Regional vote distribution
- Voter participation rates
- Gender-based voting patterns

## Technical Details

### Data Sources
- 2018 Presidential Election Results
- 2023 Presidential Election Results

### Dashboard Components
- Geographic map visualization
- Line charts for regional comparisons
- Pie charts for gender distribution
- Bar charts for candidate performance
- Comparative analysis graphs

## How to Use

1. **Navigation**
   - Use the left-side menu buttons to switch between different election years and comparisons
   - Select "2018 SEÇİMİ" for 2018 election data
     
   - Select "2023 SEÇİMİ" for 2023 election data
     ![image](https://github.com/user-attachments/assets/03ca3aae-c7e7-41ca-9327-da6fb24b47be)
     
   - Select "KARŞILAŞTIRMA" for comparative analysis
     ![image](https://github.com/user-attachments/assets/cf85c8eb-ba5d-4586-9503-7f5998a7f3ae)

2. **Map Interaction**
   - Zoom in/out using the (+/-) controls
   - Hover over regions for detailed statistics
   - Click on specific cities for detailed results

3. **Data Filtering**
   - Filter by regions
   - Filter by candidates
   - Filter by election year


## Installation

1. Clone the repository
2. Open the .pbix file with Power BI Desktop
3. Refresh data connections if needed
4. Enable automatic refresh for real-time updates
